Mob Dungeon


Mob Dungeon is a game I made in Unity using C#, as a student project for a game design class. It is a 2D top-down dungeon crawler, with combat and randomized dungeon layouts. The sprite work is outdated and not all of the systems are fully fleshed out but I will most likely return to it someday! The current build is available for download on the Github page.

Endless Runner


Based on a tutorial made for the Unity Cert Programmer Course, this is a simple endless runner, fit with powerups and sidescrolling.
